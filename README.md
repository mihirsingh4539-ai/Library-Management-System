# Library Management System

A Java-based Library Management System developed using Java, JDBC, and MySQL.

## Features

- Add Book
- View Books
- Search Book
- Update Book
- Delete Book

## Technologies Used

- Java
- JDBC
- MySQL
- VS Code
- Git & GitHub

## Project Structure

```
src/
├── App.java
├── Book.java
├── BookDAO.java
└── DBConnection.java
```
